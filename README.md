# The-French-Typst-Guide

> Guide complet Typst en français — de l'installation à la maîtrise, sous Fedora et Linux.

**Mots-clés :** Typst · guide · french

---



## Contenu

| N°   | Fascicule                   | Thèmes                                   |
| ---- | --------------------------- | ---------------------------------------- |
| 01   | Introduction & Installation | Typst, DNF/Cargo/pip, premiers pas       |
| 02   | Syntaxe de base             | Formatage, titres, listes, liens         |
| 03   | Couleurs & Styles           | Palette, surlignage, blocs callout       |
| 04   | Mise en page                | Marges, colonnes, grilles, en-têtes      |
| 05   | Tableaux & Images           | Tableaux stylisés, figures, légendes     |
| 06   | Variables & Fonctions       | let, fonctions, conditions, boucles      |
| 07   | Règles set & show           | Configuration globale, styles avancés    |
| 08   | Mathématiques & Code        | Formules, équations, blocs de code       |
| 09   | Paquets & Templates         | Typst Universe, templates réutilisables  |
| 10   | Workflow & Mémos            | Fedora 43, scripts, antisèches complètes |

---



## Fichiers

| Fichier      | Rôle                                         |
| ------------ | -------------------------------------------- |
| `GUIDE.typ`  | Contenu complet du guide                     |
| `commun.typ` | Styles partagés — **requis** par `GUIDE.typ` |

Les deux fichiers doivent être dans le **même dossier**.

---



## Utilisation

```bash
typst compile GUIDE.typ
```
